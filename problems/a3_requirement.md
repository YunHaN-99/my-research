# A3 requirement

目标：把 A3 的首发条目冻结为“带噪一维曲线拟合”，作为中期后续扩展家族的第一个正式任务，而不是继续停留在 placeholder。

## 任务定义
- 主任务：根据带噪观测点拟合连续一维曲线，并在固定评测点上给出预测结果。
- 当前冻结条目：`curve fitting`
- 问题边界：输入是训练采样点 `(x, y)` 与评测点 `eval_x`，输出是拟合结果 `pred_y` 与误差指标。

## 输入
1. `train_x`：一维自变量采样点
2. `train_y`：对应观测值，可含噪声
3. `eval_x`：待预测的评测点
4. 可选超参数：`degree / basis_type / l2_lambda`

## 输出
1. `pred_y`：在 `eval_x` 上的拟合结果
2. `fit_model`：最小可复用的拟合参数或模型对象
3. 实验产物：拟合曲线图、误差表、run 记录、metrics 条目

## 报告必须包含
1. 拟合模型假设与基函数说明
2. 训练点 / 拟合曲线 / 评测点结果图
3. `RMSE / MAE` 等误差指标
4. 对噪声、欠拟合 / 过拟合边界的简短分析
5. failure case 与误差来源说明

## 本轮暂不做
1. `A3` 的昆虫分类分支
2. 大规模模型搜索
3. Web / GUI 展示层
4. 自动化题库扩写

## 补充说明
- A3 当前只冻结 1 个首发条目，目标是把扩展家族从 placeholder 推进到“已有 requirement + taskcard_v1”。
- 昆虫分类保留为 A3 的下一批候选条目。
- 这一步的目的主要是补齐中期题库结构，而不是立刻完成 A3 的整套实验闭环。
