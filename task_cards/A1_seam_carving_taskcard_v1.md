# A1 Seam Carving Task Card v1

## 1. 题目一句话目标
实现内容感知的图像缩放：通过逐条删除低能量 seam 来缩小图像，尽量保留主体结构，而不是直接 resize 或 crop。

## 2. 输入 / 输出
输入：
- 原图 im，形状为 H x W x C。
- 目标尺寸 sz = (target_h, target_w)。

输出：
- 缩放后的图像 out，尺寸为 target_h x target_w x C。
- 实验产物：对比图、运行记录、报告内容（原理、结果、对比、说明）。

## 3. 核心模型
- 能量图 E：给每个像素定义“重要性”，能量越低越适合被删除。
- 一条 seam：从上到下的相邻像素路径；相邻指第 i 行到第 i+1 行列索引最多变化 1。
- 目标求解：用动态规划找到累计能量最小的 seam，删除该 seam，重复直到达到目标宽度。

## 4. 算法流程
1. 计算能量图。
2. 动态规划求累计最小代价。
3. 回溯得到最优 seam。
4. 删除一条 seam。
5. 重复直到达到目标宽度。
6. 高度缩放用转置复用。

## 5. 代码接口
- 主接口：seam_carve_image(im, sz)
- 建议拆分的最小函数：
	- compute_energy(im) -> E
	- find_vertical_seam(E) -> seam_idx
	- remove_vertical_seam(im, seam_idx) -> im_new
	- seam_carve_width(im, target_w) -> im_w
	- seam_carve_height(im, target_h) -> im_h（通过转置调用宽度逻辑）
- 最小实现策略：先只跑通宽度缩小，再接高度缩小。

## 6. 常见错误与自检
- 错误 1：DP 边界处理错（第一列/最后一列越界）。
	- 自检：用 5x5 小图打印 DP 表，检查边界只来自合法父节点。
- 错误 2：回溯 seam 断裂（相邻行列差超过 1）。
	- 自检：断言 abs(seam[i] - seam[i-1]) <= 1。
- 错误 3：删除 seam 后图像维度不对。
	- 自检：每删一次宽度必须减 1，最终宽度等于 target_w。
- 错误 4：高度缩放转置前后通道顺序错。
	- 自检：转置两次后应恢复原形状约定。
- 错误 5：dtype 或数值范围异常导致结果发黑。
	- 自检：输出前检查数据类型与像素范围是否与输入一致。

## 7. 评估指标与实验设计
- 结果展示：
	- 宽度缩小样例。
	- 高度缩小样例。
	- 与 crop / resize 在相同目标尺寸下的并排对比。
- 观察指标：
	- 主体保留程度（是否被切断、是否明显变形）。
	- 背景压缩自然度（是否出现明显拉扯纹理）。
	- 运行时间（不同缩放比例下的耗时）。
- 最小实验集：
	- 至少 2 张图（主体集中 + 背景复杂各 1 张）。
	- 每张图做两组缩放（改宽、改高）。
	- 每组都给出 seam carving / crop / resize 三法对比与简短结论。
