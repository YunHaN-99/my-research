# A3 Task Card v1

## 1. 题目一句话目标
实现带噪一维曲线拟合：根据训练采样点恢复一条连续曲线，并在固定评测点上输出可复查的预测结果。

## 2. 当前冻结条目
- family：A3
- 首发条目：`curve fitting`
- 当前定位：扩展家族的首个正式条目，已完成 requirement + taskcard_v1，但尚未进入完整实验闭环
- 后续备选：昆虫分类

## 3. 输入 / 输出
输入：
- `train_x`：训练采样点
- `train_y`：带噪观测值
- `eval_x`：评测点
- 可选超参数：`degree / basis_type / l2_lambda`

输出：
- `pred_y`：在 `eval_x` 上的预测结果
- `fit_model`：最小可复用的拟合参数或模型对象
- 实验产物：拟合图、误差表、run 文档与阶段记录

## 4. 核心模型
- 目标：在噪声存在时恢复尽量平滑且误差可控的拟合曲线
- 建议路线：
  - 多项式回归
  - 带 `L2` 正则的最小二乘
  - 其他等价的一维曲线拟合方法
- 中期阶段不要求模型复杂，重点是：
  - 任务边界明确
  - 输入输出可评分
  - 可自然接入结构化检查与 failure-case 设计

## 5. 最小算法流程
1. 读取训练点与评测点。
2. 选择基函数或拟合模型。
3. 在训练点上求解模型参数。
4. 在 `eval_x` 上输出预测结果。
5. 计算误差指标并保存拟合图。
6. 记录 run 文档与结果摘要。

## 6. 常见错误与自检
- 错误 1：训练点和目标值长度不一致。
  - 自检：断言 `len(train_x) == len(train_y)`。
- 错误 2：高阶多项式导致明显过拟合。
  - 自检：同时查看训练误差和评测误差，不只盯住训练集。
- 错误 3：评测点未按训练时相同方式做缩放或预处理。
  - 自检：把预处理逻辑写成单独函数并复用。
- 错误 4：只给数值，不给拟合图，导致结果不可解释。
  - 自检：每次至少输出 1 张训练点 + 拟合曲线图。

## 7. 评估指标与实验设计
- 固定指标：
  - `RMSE`
  - `MAE`
- 最小展示：
  - 训练点散点图
  - 拟合曲线
  - 评测点预测结果
- 最小实验建议：
  - 1 组中等噪声 case
  - 1 组高噪声或外推 case
